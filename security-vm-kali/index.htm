CYBER CLOAK

How good are you at hidding?

You will learn how to install a virtual machine and explore one of the most known, by security experts, distributions of linux: KALI LINUX. In addition, you will discover how to hide and unhide information from an image. This art is called steganography.

Imagine you can hide your java program in a picture and have a partner never notice it.

WORKSHOP AGENDA

  • Virtual Machine Installation
  • Kali Linux installation & Exploration
  • Introduction to Steganography
  • Challenges
    • Text file
    • Java program
    • Audio file

INSTRUCTIONS

If you would like to partcipate please make sure you come prepared. This will be a BYOL event. *Bring Your Own Laptop.

Download
  1. Virtual machine
    • for Windows users: VMWare workstation player 15.
    • for Mac users: VirtualBox
  2. The Kali Linux image files from kali.org. Make sure you select according to your personal computer specifications.

After you have downloaded the corresponding programs, here are some instrucitons to help you set up your Virtual Machine.

  • Windows & Linux users: see instructions and a youtube video to install VMware + Kali installation. There might be the case where you would have to allow virtual machines on your BIOS.
  • Mac users: see instructions and a youtube video to install virtual box and Kali Linux.
